# Exam Seating Arrangement System

This project is a web-based application for automating exam seating arrangements.

## Features
- **Admin Dashboard**: Manage students, classrooms, and exams.
- **Automated Seating**: Generate seating plans based on constraints (max 2 per bench, no same exam code).
- **Export**: Download seating charts as PDF/Excel.

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: SQLite
